# ddclient

[![Cargo](https://github.com/jbhannah/ddclient/workflows/Cargo/badge.svg)][actions-cargo]

A dynamic DNS service client in Rust.

## Supported services

- [Cloudflare][]

## Installation

    cargo install --git https://github.com/jbhannah/ddclient.git

## Usage

    ddclient help
